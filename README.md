# GitFlow-Investigacion

## Descripción
Esta actividad es una investigación técnica grupal sobre una tecnología concreta del ecosistema de desarrollo web, seguida de una demostración de código ejecutada en vivo durante la clase.

Este proyecto tiene como objetivo demostrar el uso de GitFlow mediante la creación de ramas feature, develop, hotfix y main, utilizando commits descriptivos y trabajo colaborativo en GitHub.
